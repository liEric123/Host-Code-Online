# Contributing to Host Code Online

Thanks for your interest in contributing! We welcome all contributions.

## 🤝 How to Contribute

- **🐛 Report bugs**: Found a bug? Open an issue
- **💡 Suggest features**: Have an idea? Let us know
- **🔧 Fix issues**: Pick up an existing issue
- **✨ Add features**: Implement new functionality

## 🚀 Getting Started

1. **Fork the repository**
2. **Clone your fork**
   ```bash
   git clone https://github.com/yourusername/Host-Code-Online.git
   cd Host-Code-Online
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the app**
   ```bash
   python3 app.py
   ```

## 📦 Making Changes

1. **Create a branch**
   ```bash
   git checkout -b feature/your-feature
   ```

2. **Make your changes**
   - Keep changes focused
   - Follow existing code style
   - Test thoroughly

3. **Commit and push**
   ```bash
   git add .
   git commit -m "Add your feature"
   git push origin feature/your-feature
   ```

4. **Create a Pull Request**

## 📋 Guidelines

### Code Style
- **Python**: Follow PEP 8
- **JavaScript**: Use consistent indentation
- **HTML/CSS**: Keep it clean and organized

### Commit Messages
- `feat: add new feature`
- `fix: resolve bug`
- `docs: update documentation`
- `style: improve UI`

### Testing
- Test your changes on different browsers
- Test file upload functionality
- Check both light and dark modes

## 🐛 Reporting Issues

Include:
- Your OS and browser
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if helpful

## 💡 Feature Requests

Describe:
- The problem you're solving
- How it should work
- Why it would be useful

## 🎯 Good First Issues

Look for issues labeled:
- `good first issue`
- `help wanted`
- `bug`
